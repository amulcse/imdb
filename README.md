# goodflix-style movie carousel

Left panel: movie details + search. Right panel: **3 rows** of movie posters in a round, rotatable cylinder.

**No API required.** Every movie has a real poster image from Wikipedia/Wikimedia Commons. Movies without an available poster were removed; the list includes only films with embedded poster URLs.

## Run

```bash
npm install
npm run dev
```
